Phishing Detection System

A Machine Learning and NLP-based web application that detects whether a URL is malicious (phishing) or safe. Users can enter a URL, and the system analyzes multiple URL-based features to provide real-time phishing predictions with a risk score.

🚀 Features
🔍 Real-time URL phishing detection
🤖 Machine Learning-based prediction system
🧠 NLP-inspired suspicious keyword analysis
📊 Risk score generation for URLs
🌐 Responsive frontend using HTML & CSS
⚡ REST API for URL prediction
🗂 Feature extraction for malicious URL patterns
🛠️ Tech Stack
Frontend: HTML, CSS
Backend: Python, Flask
Machine Learning: Scikit-learn
Libraries: Pandas, NumPy, Joblib, Tldextract
Model Used: Random Forest Classifier
